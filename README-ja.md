# TypoCompiler：言葉をコードとして実行するとき

**语言 / Languages / 言語 / 언어 / Idiomas / Sprachen / Langues**：  
[简体中文](./README.md) · [English](./README-en.md) · [日本語](./README-ja.md) · [한국어](./README-ko.md) · [Español](./README-es.md) · [Deutsch](./README-de.md) · [Français](./README-fr.md)


> 外国人は構文解析器ではありません。文法が間違っていても、実行を中止してはくれません。だから本物の構文解析器を作りました。

拙い外国語で話すとき、相手の笑顔の裏には「コンパイルエラー」が山ほど隠れているのでは、と考えたことはありませんか。これからは、日常会話をプログラムのようにコンパイルできます。

TypoCompiler は古典的なコンパイラのスタイルで、テキストの言語エラーを特定します。Python、Java、C++のように容赦がありません。

もう、ぎこちない笑顔の裏でこっそり `exit(1)` が呼ばれているのでは、と心配する必要はありません。

---

✨ **機能**

* **コンパイラ風の診断**：Pythonの気難しさ、Javaの厳格さ、C++の堅さ。プログラマの記憶に残る痛み。
* **多言語の自動検出**：どの言語でミスしても、TypoCompiler がきっちり指摘します。
* **クラシックなUI**：PMでも使えるほどシンプルで、エンジニアも欲しくなるほど強力。
* **LLM連携**：OpenAI互換のインターフェース。ミスを効率よく見つけ、APIクレジットも効率よく消費します。
* **スタイルのカスタマイズ**：あなたの言語、あなたのルール。社内レビューのスタイルにも柔軟に対応。

---

🧭 **はじめに**

**要件**：Python 3.8+。ほかの依存は不要です。私も怠け者なので。

```bash
python typocompiler.py
```

1. エディタを開いて、あなたの「素晴らしい」外国語を書きます。
2. LLM を設定して、AI と一緒にそのつらさを共有します。
3. **Run** をクリックして、コンパイラに容赦なく指摘してもらいましょう。

---

🖥️ **メニューガイド**

* **ファイル**：おなじみの機能。
* **設定**：言語やスタイルの切り替え。気持ちの調整は自己管理で。
* **実行**：ワンクリックで実行、クラッシュ、エラーのコピー。

---

🧠 **内蔵スタイル**

* **Python スタイル**：Traceback、定番のビンタ。
* **Java スタイル**：エラー要約、定番のお小言。
* **C++ スタイル**：文字単位まで正確、定番の辛口。

モデルが `__TC_OK__` を返したらおめでとう。少なくとも今回は AI をうまく欺けました。

---

🧩 **スタイルのカスタマイズ**

デフォルトが気に入らない？ **設定 → スタイル管理** で独自テンプレートを作って、あなたの自尊心をより正確に撃ち抜きましょう。

---

⚙️ **設定と復旧**

設定を壊してしまっても大丈夫。アプリは既定値に戻し、壊れた設定をバックアップします。

---

🌐 **プライバシーとセキュリティ**

**Run** を押すたび、あなたの言語ミスは設定した LLM サーバーへ送られます。安心してください。API キーの残高がある限りは安全です。
