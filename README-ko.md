# TypoCompiler: 언어를 코드처럼 실행할 때

**语言 / Languages / 言語 / 언어 / Idiomas / Sprachen / Langues**：  
[简体中文](./README.md) · [English](./README-en.md) · [日本語](./README-ja.md) · [한국어](./README-ko.md) · [Español](./README-es.md) · [Deutsch](./README-de.md) · [Français](./README-fr.md)


> 외국인은 문법 파서가 아닙니다. 문법이 틀렸다고 실행을 멈추지 않습니다. 그래서 진짜 파서를 만들었습니다.

서툰 외국어로 말할 때, 상대의 미소 뒤에 “컴파일 오류”가 잔뜩 숨어 있는지 궁금했던 적이 있나요? 이제 일상 대화를 프로그램처럼 컴파일할 수 있습니다.

TypoCompiler는 고전적인 컴파일러 스타일로 텍스트의 언어 오류를 집어냅니다. Python, Java, C++처럼 가차 없습니다.

더 이상 어색한 미소가 몰래 `exit(1)`을 실행하는지 걱정하지 마세요.

---

✨ **기능**

* **컴파일러식 진단**: Python처럼 까다롭고, Java처럼 엄격하며, C++처럼 딱딱합니다. 개발자의 기억 속 고통 그대로.
* **다국어 자동 감지**: 어떤 언어에서 실수하든 TypoCompiler가 정확히 집어냅니다.
* **클래식 UI**: PM도 쓸 만큼 단순하고, 엔지니어도 원할 만큼 강력합니다.
* **LLM 연동**: OpenAI 호환 인터페이스. 오류를 잘 찾아주고 API 쿼터도 잘 써줍니다.
* **스타일 커스터마이즈**: 당신의 언어, 당신의 규칙. 사내 리뷰 스타일도 쉽게 맞출 수 있습니다.

---

🧭 **빠른 시작**

**요구 사항**: Python 3.8+. 다른 의존성은 없습니다. 저도 게으르니까요.

```bash
python typocompiler.py
```

1. 에디터를 열고 당신의 “멋진” 외국어를 적습니다.
2. LLM을 설정해서 AI와 함께 그 민망함을 견뎌 봅니다.
3. **Run**을 눌러 컴파일러가 매섭게 지적하게 하세요.

---

🖥️ **메뉴 안내**

* **파일**: 익숙한 것들.
* **설정**: 언어와 스타일을 바꿉니다. 마음가짐 조절은 본인 책임.
* **실행**: 한 번에 실행, 한 번에 크래시, 한 번에 오류 복사.

---

🧠 **내장 스타일**

* **Python 스타일**: Traceback, 정통의 따끔한 맛.
* **Java 스타일**: 오류 요약, 정통의 잔소리.
* **C++ 스타일**: 문자 단위까지 정확, 정통의 팩폭.

모델이 `__TC_OK__`를 반환하면 축하합니다. 적어도 이번엔 AI를 속였네요.

---

🧩 **스타일 커스터마이즈**

기본이 마음에 들지 않나요? **설정 → 스타일 관리**에서 템플릿을 만들어 자존심을 더 정밀하게 저격해 보세요.

---

⚙️ **설정과 복구**

설정을 망가뜨렸나요? 괜찮습니다. 앱이 기본값으로 복구하고 망가진 설정을 백업합니다.

---

🌐 **개인정보와 보안**

**Run**을 누를 때마다 당신의 언어 실수는 설정한 LLM 서버로 전송됩니다. 걱정 마세요. API 키에 잔액이 있는 한 안전합니다.

---

🗂️ **개발자용**

더 깊이 파보고 싶나요? 디렉터리 구조가 잘 정리되어 있습니다. 마음껏 만져 보세요.
